대규모 네비게이션 재구성 및 신규 페이지 추가. VS Design Diverge 디자인 시스템 적용 확대.

- 하위 구조: 주보, 정착 도움 정보, 공지사항

#### 제거된 메뉴
- "섬김" (Serving) 메뉴 제거
- "자료실" (Resources) 제거

- 기도 요청
- 참여 방법

### 3. 단기선교 페이지 (`/missions/short-term`)
- 예정된 선교 여행 목록
- 진행률 표시 (목표/현재 인원)
- 4단계 준비 과정 안내
- 지난 선교 기록

### 4. 구제선교 페이지 (`/missions/relief`)
- 구제 분야 (주거, 의료, 교육, 식량)
- 최근 프로젝트
- 기부 방법
- 참여 안내

### 5. State College 정착 정보 (`/community/settlement-info`)
- **6개 카테고리**:
  - 주거/부동산 (Apartments.com, Zillow, PSU Housing)
  - 학교/교육 (SCASD, Penn State, 한국학교 연결)
  - 의료/건강 (Mount Nittany, Penn State Health)
  - 쇼핑/마트 (Wegmans, H Mart, Walmart, Costco)
  - 행정 절차 (PennDOT, SSA, USCIS)
  - 교통 (CATA Bus, Megabus, 공항)
- 정착 체크리스트 (첫 주, 첫 달, 팁)
- 모든 링크 실제 작동 확인됨
