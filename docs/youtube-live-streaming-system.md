## 오디오 시스템 연결 방법

### 방법 1: 직접 연결
```
무선 Shure 마이크 → 오디오 인터페이스 → USB → Mac Mini/아이폰
```

### 방법 2: 믹서 경유 (권장)
```
무선 Shure 마이크 → 오디오 인터페이스 → 믹서 → USB → Mac Mini
```

#### 믹서 사용의 장점
- 실시간 음량 조절
- EQ 및 이펙트 적용
- 다중 마이크 믹싱
- 모니터링 기능

## 스트리밍 설정 가이드

### YouTube 라이브 설정
1. YouTube Studio → 라이브 → 시작하기
2. 스트림 키 복사
3. OBS/Larix에 스트림 키 입력
4. 해상도: 1080p, 비트레이트: 4500 kbps

### OBS Studio 설정 (Mac Mini)
```
비디오 설정:
- 해상도: 1920x1080
- FPS: 30
- 인코더: H.264 (하드웨어)

오디오 설정:
- 샘플레이트: 48kHz
- 비트레이트: 160 kbps
```

### 네트워크 요구사항
- 업로드 속도: 최소 10Mbps (권장 20Mbps)
- 유선 인터넷 연결 권장
- 백업 인터넷 준비 (핫스팟)

## 카메라 앵글 및 구성

### 기본 앵글 (Single Camera)
- 메인 카메라: 강대상 정면
- 높이: 설교자 눈높이
- 거리: 2-3미터

### 다중 앵글 (Multi Camera)
1. **메인 샷**: 강대상 전체 (Wide)
2. **클로즈업**: 설교자 상반신 (Medium)
3. **회중 샷**: 예배 참석자들 (옵션)

## 조명 설계

### 기본 3점 조명
1. **키 라이트**: 설교자 정면 45도
2. **필 라이트**: 반대편 보조 조명
3. **백 라이트**: 배경과 분리용

### 조명 배치도
```

    [설교자]
   /         \
[키라이트]  [필라이트]

    [카메라]
```
